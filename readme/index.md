# 本サーバーについて

本サーバーについての約束事やこうしますというものを記載します。  
なるべく皆が過ごしやすいサーバーであり続けるため、サーバー参加者の努力に期待をします。

## こころ避難所のあらまし

前身のサーバーでサーバーの喪失を経験したため、サーバーの閉鎖を行わない事に注力します。（discord運営元から削除されない限り消さない）  
活動内容は主にVCやチャットでの雑談となります。  

## 本サーバーの対象者

- メンタルを抱えている人
- セクシャルマイノリティ
- ぼっち/不登校/ひきこもり/ニート
- 上記に対して差別を行わない人

なお、参加した場合にはあなたもサーバーの一員です。  
お客様ではないので、その点ご留意ください。

## サーバーでのおすすめな活動のやり方

1. \#自己紹介で自己紹介文を書く
1. 人がいたらそのVCに挨拶をしてはいる。  [マナー](#マナー)
1. 雑談であいさつをする。
1. 人がいなかったら、\#募集に「@here VCチャンネル名 募集」として人を募集する。

## 禁則事項と措置について

- 上記の人を侮蔑する行為
- ハラスメント・荒らし等の行為
  - 無差別の DM or フレンド登録
- 会話が成立しないなどコミュニケーションを取ることができない場合
  - 愚痴やネガティブな事を吐き出してばかりの場合
- 他コミュニティへの移動宣言・本サーバーの宣伝行為
- 医療行為への過度なアドバイス
- 自傷行為等
  - 自傷行為の画像・動画のアップロード
  - ボイスチャンネルでの自傷行為
- 管理人判断
  - 苦情時に措置必要と判断した場合
  - 協議・判断の結果、措置必要と判断した場合
- 本サーバーはKICKいかんにかかわらず、合計で３回抜けると自動的にBANになります。

理由を記載の上、KICK/BAN対応を行います。  
理由は、\# join_listにて確認できます。 KICKの場合には戻ったあと、理由をご確認ください。

### 苦情等

- 基本的に対人関係にて生じたことは当人同士で解決を試みてください。
  - どうしても解消できない場合は、[苦情テンプレート](./claim.md)をコピペ後、記入した結果を管理人へDMください。
    - ちゃんと見てわかるように簡潔に記載してください。内容次第でアクションを起こします。
    - 読んでも分からない場合や内容で管理人が対処するべきではないと判断した場合は、アクションを起こさない場合もありますがご了承ください☆
    - VCで苦情言われても管理人はそのままスルーするかもしれません。（管理人が苦情テンプレートを元にドキュメントを作って別途管理人達に展開する労力は無駄なため）
- DMでの個人間のやりとりについての苦情は受け付けません。(禁則事項を除く)

## 主要チャンネルの説明

### 自己紹介（任意）

任意であなたのことでみんなに教えてもいい範囲内で書いてもいいかな？と思ったら書いてください。  
話すときにこちらを見て、話する人がいますので、書いておいた方が若干楽かも。  
自己紹介テンプレート:

```text
【名前】
【趣味】
【年代】
【カテゴリ】
【みんなへの一言】
【その他】
```

### 募集

VCで話したい人を募集するときに、募集チャンネルにて「@here 通話募集」みたいな感じで通話相手を募集してください。  
VCに誰もいなくても、ワラワラと集まる可能性があります。

## マナー

### VCに入るとき

- ハウリングしないようにね。ハウリング指摘されたら、ミュートにして設定してからまた入ってね。

### 心得

- フレンドのみDM許可することを推奨します
- いきなりのDM・フレンド申請はあやしいのでBANリストに突っ込みましょう。
  - そして、管理人に報告してくれると助かります。

### 管理人心得

- 無理しない、続けることこそ善
- リアル優先
- 無駄に気張らない、管理人は問題が起こったら相談するだけでいいです。　なんとなく決まったことで仮にその人がBANになった場合でも、あなたの責任ではありません。
